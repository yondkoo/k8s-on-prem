  - `./etcd-instances` дотор `etcd-1` ажиллана.
  - `./keepalived-ha-instances` дотор `lb1` ажиллана.
  - `./master-instances` дотор `kmaster1`, `kmaster2` ажиллана.
  - `./worker-instances` дотор `kworker1` ажиллана.
